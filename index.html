Project 01

Automated
Class Engine Scheduling
System

Developed by: Kurt Michael S. Salut

Also known as A.C.E., this web-based system is designed to help administrators and instructors at Biliran Province State University efficiently manage class schedules. The system automates tasks such as assigning instructors, subjects, rooms, and time slots, thereby minimizing conflicts, reducing errors from manual scheduling, and optimizing the use of available resources.

Project 02

Automated Student Class Record Monitoring
Web Application

Developed by: Arian Jay G. Maala

Also known as ASCReM, this web application is designed to help instructors and administrators at the School of Technology and Computer Studies (STCS) manage student records more efficiently. It automates tasks such as attendance tracking, score input, grade computation, and report generation, thereby reducing manual work, improving accuracy, and enabling real-time access.
